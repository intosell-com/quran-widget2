# ویجت تصادفی پوسترهای قرآنی

یک ویجت فوق‌مدرن، سبک و بدون سرور برای نمایش تصادفی پوسترهای قرآنی در سایت، وبلاگ یا GitHub Pages.

این پروژه با فایل متنی `quran.txt` کار می‌کند. هر خط این فایل یک آدرس مستقیم تصویر است و برنامه در هر بار بارگذاری، یکی از تصاویر را به‌صورت تصادفی نمایش می‌دهد.

## ویژگی‌ها

- نمایش تصادفی تصویر از روی فایل `quran.txt`
- قابل انتشار روی GitHub Pages
- قابل استفاده در سایت‌ها و وبلاگ‌های دیگر با یک کد اشتراک‌گذاری
- طراحی واکنش‌گرا و مناسب موبایل، تبلت و دسکتاپ
- بدون نیاز به دیتابیس، پنل مدیریت یا سرور اختصاصی

## فایل‌ها

- `index.html`: صفحه اصلی، پیش‌نمایش و کد اشتراک‌گذاری
- `embed.html`: نسخه سبک مخصوص نمایش داخل سایت‌های دیگر
- `widget.js`: اسکریپت اشتراک‌گذاری برای سایت‌ها و وبلاگ‌ها
- `app.js`: منطق خواندن `quran.txt` و نمایش تصادفی تصویر
- `styles.css`: طراحی فوق‌مدرن و واکنش‌گرا
- `quran.txt`: فهرست آدرس تصاویر آیات و پوسترهای قرآنی


## کد استفاده در سایت‌های دیگر

بعد از انتشار پروژه روی GitHub Pages، این کد را در سایت یا وبلاگ مقصد قرار دهید:



به جای `USERNAME` نام کاربری GitHub و به جای `REPOSITORY` نام مخزن خودتان را قرار دهید.

## کد جایگزین iframe

اگر سایت مقصد اجازه اجرای اسکریپت نداد، می‌توانید از نسخه iframe استفاده کنید:

```html
<iframe src="https://USERNAME.github.io/REPOSITORY/embed.html" style="width:100%;max-width:900px;height:470px;border:0;border-radius:34px;overflow:hidden" loading="lazy"></iframe>
```

## تنظیمات اختیاری اسکریپت

```html
<script
  src="https://USERNAME.github.io/REPOSITORY/widget.js"
  data-quran-widget
  data-height="470"
  data-max-width="900px"
  data-radius="34px">
</script>
```

## نکته مهم

برای اجرای درست پروژه، فایل `quran.txt` باید در کنار فایل‌های اصلی پروژه و در همان مسیر انتشار قرار داشته باشد.
